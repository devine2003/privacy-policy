GameNest – Delete Account Policy

App Name: GameNest
Package Name: com.krisha.g.gamenust

1. Account Deletion Overview

Users can request deletion of their GameNest account at any time. Once deleted, all user-related data will be permanently removed.

2. Data That Will Be Deleted

When a user deletes their account, the following data will be removed:

  • Username and profile data
  • Game progress and scores
  • Multiplayer session history
  • Stored preferences and settings

This action is permanent and irreversible.

3. How to Request Account Deletion

Option 1: From the App

  • Open Settings
  • Select Delete Account
  • Confirm deletion

Option 2: Via Email

Send an email to: zenskarsoftware@gmail.com

Include:

  • Username
  • Reason for deletion (optional)

4. Processing Time

Account deletion requests are processed within 3–7 working days.

5. Data Retention

  • No personal data is retained after deletion.
  • Anonymous (non-identifiable) data may be used for analytics purposes.

6. Contact

For any concerns regarding account deletion, contact: zenskarsoftware@gmail.com
