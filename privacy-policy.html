GameNest – Privacy Policy

App Name: GameNest

Package Name: com.krisha.g.gamenust

Last Updated: 05-01-2026

1. Introduction

GameNest is an all-in-one casual gaming platform that allows users to play multiple games from a single application. We respect your privacy and are committed to protecting your personal information.

This Privacy Policy explains how we collect, use, and protect user data while using the GameNest application.

2. Information We Collect

2.1 User Information

  • Username (used for gameplay identity)
  • Game preferences and in-app settings

2.2 Game Data

  • Game progress and scores
  • Multiplayer game data (room ID, turns, win/loss status)
  • Real-time interactions during gameplay

2.3 Device Information

  • Device model and OS version
  • App performance data (for crash fixes and improvements)

3. Permissions Used

GameNest uses the following permissions for proper functionality:

  • INTERNET – Required for online gameplay and real-time updates
  • MODIFY_AUDIO_SETTINGS – To manage in-game sound controls
  • RECORD_AUDIO – For voice features (if enabled in future updates)
  • READ_EXTERNAL_STORAGE / WRITE_EXTERNAL_STORAGE – For saving game data and assets
  • SYSTEM_ALERT_WINDOW – Used for in-game overlays or alerts
  • VIBRATE – For haptic feedback during gameplay

We do not access personal files or sensitive data without user consent.

4. How We Use Your Data

Your data is used only for:

  • Game functionality
  • Multiplayer synchronization
  • Improving gameplay experience
  • Maintaining session continuity

We do not sell, rent, or share user data with third parties.

5. Multiplayer & Real-Time Communication

GameNest uses Socket.IO for:

  • Real-time gameplay
  • Player synchronization
  • Turn-based actions
  • Live game status updates

All communication is securely handled and used only during gameplay.

6. Data Security

We use standard security practices to protect your data. However, no system is 100% secure, and users are encouraged to play responsibly.

7. Children’s Privacy

GameNest does not knowingly collect personal data from children under the age of 13.

8. Changes to This Policy

We may update this Privacy Policy from time to time. Any updates will be reflected within the app or on the app listing page.
